# Portfolio Site

This is a static personal portfolio site for GitHub Pages.

## Customize content

Edit `profile.js` to replace the sample achievements, metrics, contact details, LinkedIn URL, and GitHub URL with your real information.

## Preview locally

From the `portfolio-site` folder, run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish with GitHub Pages

1. Create a GitHub repository for this folder.
2. Push the contents of `portfolio-site` to the repository root.
3. In GitHub, enable Pages with `GitHub Actions` as the source.
4. The included workflow in `.github/workflows/deploy.yml` will publish the site automatically on each push to `main`.

## Notes

- The current content is strong starter copy designed to emphasize leadership and transformation work.
- Replace the sample metrics with exact numbers before using it publicly.
